// l'architecture

Orchestrated. Not scripted.

LeClaw runs as a coordinated agent hierarchy. Le Directeur dispatches une mission. Each agent files un rapport. Le Témoin guards every write-back.

LD
Le Directeur
orchestrateur · dispatches une mission
The orchestrating agent. Coordinates which agents run, in what order, with what context. Reads every rapport and synthesizes a final insight no individual agent can see.
dispatches → une mission
13
les agents
spécialistes · each files un rapport
Each agent owns a domain: data quality, pipeline stage, BDR follow-up, forecast. They fetch only broken records from your CRM — no full scans, no wasted compute. Every agent returns a structured rapport: a score, issues list, and AI summary. If a check fails fatally, it exits via Le Retrait without tainting the mission.
proposes → write-back
LT
Le Témoin
validateur · reviews before write-back
No change touches your CRM without passing through Le Témoin. Approval can be automatic (high-confidence, low-risk) or require human sign-off. Write-back is always opt-in.
une mission
A coordinated run dispatched by Le Directeur. Multiple agents, one shared context.
un rapport
Structured result from each agent: score, issue list, AI summary. Stored as a Bead.
Le Retrait
Fail-fast exit. A check that cannot complete exits cleanly without corrupting the mission.
